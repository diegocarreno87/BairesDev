******************************************************
Please read the following before running the solution:
******************************************************
1.1 If you have an Existing database available to test, 
	please run the BDev.sql script on that database.
1.2 If you do not have an existing database created, 
	please create one or run the BDev-Database.sql script to create a new one.
	Then run the BDev.sql script.
2.1 On the BDev.Web project, please update the connection string with
	the right server-User-Password so the solution can have a
	successful database login.
3.1 Please validate the dbo.BDevUsers table, you can delete-add new records there.
	The logon screen is using dbo.BDevUsers table to validate the
	Username and Password.
******************************************************
The Solution is using MVC on the front-end 
All the content is separated on 6 layers.
